# Anima Theme Images

## Source
Copy images from: https://github.com/abdullahshioncse/anima/tree/main/img

## Required Images

### Icons (All iconly-sharp-*.svg files)
Key icons needed:
- `iconly-sharp-profile.svg` - User profile icon
- `iconly-sharp-bag-2.svg` - Shopping bag icon
- `iconly-sharp-heart.svg` - Wishlist/heart icon
- `iconly-sharp-search.svg` - Search icon
- `iconly-sharp-arrow---left-2-1.svg` - Left arrow for carousel
- `iconly-sharp-arrow---right-2.svg` - Right arrow for carousel
- `iconly-sharp-bag-2-1.svg` - Info bar bag icon
- `iconly-sharp-time-circle.svg` - Delivery time icon
- `iconly-sharp-tick-square.svg` - Authenticity check icon

### Dividers
- `line-1-1.svg`, `line-2-1.svg`, etc. - Decorative divider lines

### Hero Banner
- `rectangle-5-1.png` - Main hero banner image

### Payment Icons
- `mastercard-1@2x.png` - Mastercard logo
- `visa-logo-1.svg` - Visa logo
- `applepay-1.svg` - Apple Pay logo

### Product Images (Examples)
- `rectangle-1166.png` - Product image 1
- `rectangle-1167.png` - Product image 2
- `rectangle-1168.png` - Product image 3
- `rectangle-1169.png` - Product image 4

### Other Icons
- `star-5.svg`, `star-6.svg`, `star-7.svg` - Rating stars
- `add.png` - Add to cart icon
- `minus.png` - Decrease quantity icon

## Installation
1. Download all images from the anima repository
2. Copy them to this directory: `catalog/view/theme/journal3/image/anima/`
3. Maintain the original filenames

## Note
Images are not included in the repository to reduce size. They must be copied from the source repository before the theme will display correctly.
