# Anima Theme - Required Image Assets

This directory should contain the following image assets from the Anima design repository.

## Required Icons (SVG format)

Place these icon files in this directory:

### Navigation & Interface Icons
- `iconly-sharp-search.svg` - Search icon
- `iconly-sharp-profile.svg` - User account icon
- `iconly-sharp-heart.svg` - Wishlist/favorite icon
- `iconly-sharp-bag.svg` - Shopping cart icon
- `iconly-sharp-call.svg` - Phone/contact icon
- `iconly-sharp-show.svg` - Quick view/eye icon

### Info Bar Icons
- `iconly-sharp-paper.svg` - Returns/document icon
- `iconly-sharp-delivery.svg` - Delivery/shipping icon
- `iconly-sharp-shield-done.svg` - Authenticity/security icon

## Payment Method Logos

Place these payment logo files in this directory:

- `visa.svg` - Visa card logo
- `mastercard.svg` - Mastercard logo
- `applepay.svg` - Apple Pay logo

## Placeholder Images

If you need placeholder images for testing:

- `placeholder-product.jpg` - Product placeholder (recommended size: 500x500px)
- `hero-banner.jpg` - Hero banner image (recommended size: 1920x600px)

## Line Separators (Optional)

- `line-*.svg` - Various decorative line separators used in the Anima design

## Image Specifications

### Icons
- Format: SVG (Scalable Vector Graphics)
- Color: Black (#000000) - will be styled via CSS
- Size: Should be designed for 24x24px base size

### Payment Logos
- Format: SVG preferred, PNG acceptable
- Size: Height should be ~30px, width proportional

### Product Images
- Format: JPG or WebP
- Recommended size: 800x800px minimum
- Aspect ratio: 1:1 (square)

### Hero Banner
- Format: JPG or WebP
- Recommended size: 1920x600px
- Aspect ratio: 16:5

## How to Use

1. Download the Anima design repository or assets
2. Copy all the icon files listed above to this directory
3. Ensure file names match exactly as listed
4. If using custom icons, maintain the same naming convention

## Notes

- All icons should maintain a consistent style (outlined, sharp corners as per Iconly Sharp set)
- SVG format is preferred for scalability and performance
- Images should be optimized for web (compressed)
- Consider providing 2x versions for retina displays

## Alternative Icon Sets

If Iconly Sharp icons are not available, you can use similar icons from:
- Feather Icons (https://feathericons.com/)
- Heroicons (https://heroicons.com/)
- Font Awesome (https://fontawesome.com/)

Just ensure to maintain consistent styling and update the file names accordingly in the theme files.
